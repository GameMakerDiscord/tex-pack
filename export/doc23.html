Texture pages

A texture page is a top-level object that holds sprites and texture pages inside.

new TexPage(w, h)

Creates a new, empty texture page of specified size.

width:number

Read-only; holds the width of this texture page

height:number

Read-only; holds the height of this texture page

root:TexEntry

Holds the top-level texture page entry node

sprite:sprite

Once finalized, holds the generated sprite

sprites:array<TexSprite>

An array of sprites belonging to this texture page.

surface:surface

The temporary surface holding the sprites. Is destroyed on finalize.

destroy()

Destroys the texture page and any associated resources.
Don't forget to destroy your texture pages when you are done using them!

add(fname, imgNumb, ox, oy)➜TexSprite

Basically sprite_add, but for texture pages.

finalize()

Creates the actual texture page sprite and destroys the temporary surface.

You need to finalize before you can draw sprites.

Sprites

Represents a sprite on a texture page.

new TexSprite(count, xofs, yofs)

Generally you shouldn't have to instantiate sprites manually.

images:array<TexEntry>

An array of texture page entries for each sprite frame.
If something couldn't fit, an item may be undefined.

count:int

(read-only) The number of subimages.

sprite:sprite

Once finalized, holds the actual (GameMaker) sprite.

xoffset:number

(read-only) Sprite's origin X.

yoffset:number

(read-only) Sprite's origin Y.

setOffset(xo, yo)

Updates the origin of the sprite.

Drawing functions:

draw(subimg, x, y)

Equivalent of draw_sprite

drawExt(subimg, x, y, scaleX, scaleY, rot, col, alpha)

Equivalent of draw_sprite_ext

drawPart(subimg, left, top, width, height, x, y)

Equivalent of draw_sprite_part

drawPartExt(subimg, left, top, width, height, x, y, sx, sy, rot, c, a)

Equivalent of draw_sprite_part_ext, except with angle, because why not.

drawGeneral(subimg, left, top, width, height, x, y, sx, sy, rot, c1, c2, c3, c4, a)

Equivalent of draw_sprite_general

Texture page entries

Represents an individual image on a texture page.

new TexEntry(x, y, w, h)

The following define a rectangle within the texture page:

left:number
top:number
width:number
height:number

The following define the subimage's origin:

xoffset:number
yoffset:number

The following hold the child nodes (if any):

nodeA:TexEntry
nodeB:TexEntry
add(imgWidth, imgHeight)➜TexEntry

Tries to allocate a spot of given dimensions in this or child nodes.

Returns the texture page entry that was chosen.

If there is no free spot, returns undefined.
